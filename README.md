# ☁️ ListenUp - English Learning for Slow Learners

An interactive English listening app designed specifically for slow learners with cloud audio theme and female voice support.

![License: MIT](https://img.shields.io/badge/License-MIT-green)
![Version: 1.0](https://img.shields.io/badge/Version-1.0-blue)

## 🌟 Features

- **👩 Female Voice** - Optimized speech synthesis for slow learners
- **🐢 Slow Learning Mode** - Adjustable speeds from 0.5x to 1.5x
- **🔁 A-B Loop** - Repeat any section for practice
- **☁️ Cloud Theme** - Beautiful dark UI with floating animations
- **📚 10 Lessons** - From Beginner to Advanced
- **📊 Progress Tracking** - Track your learning journey

## 📖 How to Use

1. **Select a lesson** from the Library
2. **Choose voice** - Female (slower) or Male
3. **Adjust speed** - Start with 0.5x or 0.75x
4. **Use A-B loop** to repeat difficult sections
5. **Click "Speak Transcript"** to hear the text

## 🛠️ Built With

- HTML5, CSS3, JavaScript
- Tailwind CSS
- Web Speech API
- LocalStorage for progress
